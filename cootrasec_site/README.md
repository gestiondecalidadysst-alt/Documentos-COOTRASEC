# COOTRASEC - Repositorio Documental (Static Site)

Archivos listos para subir a GitHub Pages.

**Archivos incluidos**:
- index.html
- styles.css
- scripts.js
- favicon.png

**Instrucciones rápidas**:
1. Reemplaza en `index.html` el valor `AQUI-VA-TU-LINK-DE-LOGO` por el URL público de tu logo (o súbelo a la carpeta `assets/` y referencia `assets/logo.png`).
2. Reemplaza los enlaces `LINK-ONEDRIVE-*` por los enlaces públicos de OneDrive.
3. Crea un nuevo repositorio en GitHub y sube estos archivos.
4. En Settings → Pages, selecciona la rama `main` y la carpeta `/ (root)` y guarda. Después de unos minutos tu sitio estará disponible en `https://TU-USUARIO.github.io/TU-REPO`.

**Notas**:
- El sitio es totalmente estático. Para autenticación o control de acceso hace falta backend.
- Si quieres, te puedo generar el ZIP listo para subir o subirlo yo a un repo público si me das el permiso y nombre de repo.
